ARCHIVOS INCLUIDOS
==================

  📦 NetBoozt_GUI.exe    Interfaz gráfica con icono en bandeja del sistema
  📦 NetBoozt_CLI.exe    Versión de línea de comandos (terminal)


CUÁL USAR
=========

  NetBoozt_GUI.exe (Recomendado para usuarios)
  ---------------------------------------------
  • Interfaz visual moderna
  • Icono junto al reloj de Windows
  • Monitoreo en tiempo real
  • Click derecho en tray para acciones rápidas

  NetBoozt_CLI.exe (Para usuarios avanzados)
  ------------------------------------------
  • Menú interactivo en terminal
  • Ideal para servidores o scripts
  • Menor uso de recursos
  • Sin dependencias gráficas


CÓMO EJECUTAR
=============

  1. Click derecho en el ejecutable
  2. Seleccionar "Ejecutar como administrador"
  
  (Se requieren permisos de administrador para cambiar DNS y 
   aplicar optimizaciones TCP/IP)


CARACTERÍSTICAS v2.2
====================

  ✓ DNS Intelligence - Selección automática del mejor DNS
  ✓ System Tray - Icono junto al reloj con menú
  ✓ Diagnósticos de red en 4 fases
  ✓ Monitoreo de Windows Event Log
  ✓ Optimizaciones TCP/IP (BBR-like)
  ✓ Auto-failover entre 8 DNS servers
